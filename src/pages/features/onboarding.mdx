Define the resources once and get the results on Android and iOS platforms
Change this in shared module : 

``` 
val allSwipeDatas = listOf(
    OnboardingItem( 
        title = "Choose a sketch",
        subtitle = "use your camera or gallery",
        button = "Next",
        lottie = "sketch"
    ),
    OnboardingItem( 
        title = "Describe it",
        subtitle = "be as specific as possible",
        button = "Next",
        lottie = "paper"
    ),
    OnboardingItem( 
         title = "Let the magic happen",
         subtitle = "and just look at the masterpiece",
         button = "Next",
         lottie = "magic"
     ),
     ...
)
```
