The components below are sorted alphabetically.

## BackIcon

A simple back icon to use in top of your screens 

```kotlin
fun BackIcon(navigateBack: () -> Unit)
```

Example : 
```kotlin
BackIcon {
    navController.navigateUp()
}
```

![Kickstarter Jetpack Compose Back icon component in AppKickstarter](backicon.png)

## BasicToggle

A basic toggle with a good scale and KickstarterTheme applied.

```kotlin
fun BasicToggle(
    checked: Boolean,
    modifier: Modifier = Modifier,
    onCheckedChange: (Boolean) -> Unit,
)
```

Example : 
```kotlin
var checked by remember { mutableStateOf(false) }
BasicToggle(checked) { checked = !checked }
```

## CircleImage

A simple circle image with pre-built multiplatform image rendering, fetching, caching, etc built on top of [compose-imageloader](https://github.com/qdsfdhvh/compose-imageloader).

```kotlin
fun CircleImage(url: String, modifier: Modifier = Modifier)
```

Example : 
```kotlin
CircleImage("https://i.pravatar.cc/150?u=a042581f4e29026704d")
```


## DefaultBadge

A text with a nice rounded backgroud color. KickstarterTheme applied.

```kotlin
fun DefaultBadge(
    text: String,
    modifier: Modifier = Modifier,
    backgroundColor: Color = KickstarterTheme.colors.sheetBackground,
    textColor: Color =  KickstarterTheme.colors.textOnBackground,
)
```

Example : 
```kotlin
DefaultBadge("Hello")
```

## IconWithBackgroundColor

An icon with a background color.

```kotlin
fun IconWithBackgroundColor(
    imageVector: ImageVector,
    colorIcon: Color,
    modifier: Modifier = Modifier,
    innerPadding: Dp = 8.dp,
    shape: Shape = CircleShape,
    tintIcon: Color = Color.White,
)
```

Example : 
```kotlin
IconWithBackgroundColor(Icons.Rounded.Home, Color.Blue)
```


## PressableButton

A button with specific design and which reacts to press event. KickstarterTheme applied

```kotlin
fun PressableButton(
    text: String,
    onClick: () -> Unit,
    modifier: Modifier = Modifier,
    enabled: Boolean = true,
    pattern: ImageBitmap? = null
)
```

Example : 
```kotlin
PressableButton("Press me") { 
    // do something on click
}
```
